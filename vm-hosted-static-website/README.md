# VM Hosted Static Website

This experiment demonstrates hosting a static website on
virtual machines using infrastructure provisioning tools.

## Objectives
- Provision a virtual machine
- Install and configure a web server
- Serve a static website
- Compare setup across operating systems

## Platforms Used
- macOS VM
- Windows VM
- Vagrant (VM provisioning)

## Notes
This experiment was built incrementally and reflects practical
infrastructure learning across platforms.
